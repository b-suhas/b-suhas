### Featured Projects

**[ros2-robot-monitor](https://github.com/b-suhas/ros2-robot-monitor)**
&nbsp;·&nbsp; `ROS2` `Python` `Custom Interfaces`

A two-package ROS2 workspace with a custom `RobotStatus` message interface and OOP-based publisher/subscriber nodes for real-time robot telemetry. Demonstrates typed message design, `ament_cmake` + `ament_python` workspace structure, and node graph visualization with `rqt_graph`.

---

**[ros2-turtlesim-geometry-drawer](https://github.com/b-suhas/ros2-turtlesim-geometry-drawer)**
&nbsp;·&nbsp; `ROS2` `Python` `Motion Control`

An interactive ROS2 application that autonomously draws geometric patterns in turtlesim using velocity-controlled motion, timer-driven publishers, and shape algorithms. Covers real-time robot motion, `geometry_msgs/Twist`, and clean OOP node architecture.

---

**[tuntun-mausi-voice-assistant](https://github.com/b-suhas/tuntun-mausi-voice-assistant)**
&nbsp;·&nbsp; `Python` `SpeechRecognition` `pyttsx3`

A wake-word-activated Python voice assistant with Hinglish responses, browser automation, and offline TTS. Built as a standalone systems project demonstrating speech pipelines, modular command routing, and real-world Python integration.
